Скрипты выполненного задания
- dbscript.sql - скрипт создания БД в Postres, таблицы и учетной записи для работы с таблицей
- cbrf.py - скрипт с функция загрузки котировок USD_RUB с сайта ЦБРФ
- database.py - скрипт для обращения к БД
- sheetstodb.py - скрипт с основным функционалом программы.

Для выполнения операций необходимо создать БД на такущей машине и запустить скрипт sheetstodb.py
